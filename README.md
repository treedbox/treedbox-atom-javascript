## Example
for **html** or **js** files:ls

Type **fetch** and press "Tab/Enter" to generate:
```
fetch(url)
   .then((response) =>{
     return response;
 })
   .then((data) =>{
     console.log('data', data);
 })
   .catch((error) =>{
     console.log('ERROR: ', error.message);
 })
 ```
